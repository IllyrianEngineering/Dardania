◆ abortTransaction()

Acad::ErrorStatus AcDbTransactionManager::abortTransaction ( )
pure virtual

Terminates the current (top) transaction.

Begins an immediate rollback of all changes to objects since the start of the current transaction. This rollback is applied for all transaction-resident objects.

Returns
Acad::eOk if successful,
Acad::eNoActiveTransactions if no transaction is currently running

Adds a newly created (not closed previously) object to the topmost transaction.

This is a preliminary function implementation. It allows to add to transaction a database object, which was created/added to database AFTER a transaction has been started.

Parameters

[in]addItnot used
Returns
