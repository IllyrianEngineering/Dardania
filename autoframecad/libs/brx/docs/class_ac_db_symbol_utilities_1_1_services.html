Validates a string as a symbol name.

Parameters
[in]symbolNamesymbol name to be validated
[in]isNewboolean value indicating if the symbol name is to be validated as new or existing symbol name
[in]allowVertBarboolean value indicating if vertical bars are allowed in the name
[in]isCompModeboolean value indicating if compatibility mode will be used. In compatibility mode the symbol name will be validated using extended symbol names rules. Otherwise, the name will be validated using legacy symbol name rules.
Returns
Acad::eOk if the symbol name is validated Acad::eInvalidInput if the symbol name is not validated Acad::eStringTooLong if isCompMode is false and string exceeds 31 characters
