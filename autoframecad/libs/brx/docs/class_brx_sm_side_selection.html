UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmSideSelection::setFace ( const AcDbFullSubentPath &  refFace)

Sets the reference face for the side selection operation; all faces that lie on the same side as the reference face will be selected by a successful 'run()' call;.

Parameters
[in]refFace- the reference face used by the side selection operation; 'refFace' can be a face of one of these features : a) bend b) flange c) form feature d) lofted bend e) wrong bend f) wrong flange
Returns
Acad::eOk if the input face is accepted, error status otherwise
