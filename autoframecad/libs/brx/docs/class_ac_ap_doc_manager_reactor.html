UX_EXPORT_FUNCTION void AcApDocManagerReactor::documentLockModeChanged ( AcApDocument *  pDoc,
AcAp::DocLockMode  myPreviousMode,
AcAp::DocLockMode  myCurrentMode,
AcAp::DocLockMode  currentMode,
const ACHAR *  pGlobalCmdName 
)
virtual

This function is called, when the document's locking mode has been changed; can be answered with veto(), to cancel the command, which is about to start.

Parameters
[in]pDocdocument who's locking mode has been changed
[in]myPreviousModethe highest lock mode before the change was made
[in]myCurrentModethe highest new mode for the current execution context
[in]currentModethe highest locking mode for all execution contexts
[in]pCmdNamethe command which is about to start
