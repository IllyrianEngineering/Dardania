UX_EXPORT_FUNCTION Adesk::Boolean AcGeCurve3d::area ( double  startInterv,
double  endInterv,
double &  area,
const AcGeTol &  tol = AcGeContext::gTol 
) const

Calculates the area of the curve between startParam and endParam.

On success returns Adesk::kTrue, otherwise returns Adesk::kFalse. In case curve isn't closed, a line segment will connect the opened curve and function will return the respective area. Parameters can have negative values too.

Parameters
[in]startIntervstart interval of curve
[in]endIntervend interval of curve
[out]areaobtained area of the curve defined between that interval
[in]toltolerance
Returns
On success returns Adesk::kTrue, otherwise returns Adesk::kFalse if curve isn't planar.
