Sets the polyline with values taken from a given AcDb2dPolyline.

If bDbR is true, the polyline vertices positions, objectId, handle, extended entity data, extension dictionary, and reactors becomes the same as the source 2dPolyline. The polyline is then made database resident and the source 2dPolyline is deleted.
If bDbR is false, the polyline acquires vertices positions. It stays non-database resident and 2dPolyline stays database resident.

Parameters
[in]pPlinesource AcDb2dPolyline
[in]bDbRtrue to do a handOverTo() between the source AcDb2dPolyline and this AcDbPolyline
Returns
Acad::eOk if successful,
Acad::eInvalidInput if bDbR == true and pPline is not database resident
Acad::eIllegalEntityType if pPline is not an AcDb2dPolyline
Acad::eAlreadyInDb if the calling polyline is already database resident
Acad::eNotApplicable if pPline is not a k2dSimplePoly or k2dFitCurvePoly or there is extended entity data attached to any vertex
