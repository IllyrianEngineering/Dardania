Verifies if the crossSectionCurves parameter are valid to create a solid with loft operation.

Parameters
[in]crossSectionCurvesLofting operation is performed through the set of cross section curves which are contained in this array. Each of those curves defines a cross section through which the lofted solid will pass. This parameter is mandatory.
[out]isOpenif all of the cross-section curves are open this parameter is true
[out]isClosedif all of the cross-section curves are close this parameter is true
[out]isAllPlanarif all of the cross-section curves are planar this parameter is true
Returns
Acad::eOk if the parameter crossSectionCurves is valid or invalidInput otherwise

Verifies if the crossSectionCurves and pathCurve parameters are valid to create a solid with loft operation.

Parameters
[in]crossSectionCurvesLofting operation is performed through the set of cross section curves which are contained in this array. Each of those curves defines a cross section through which the lofted solid will pass. This parameter is mandatory.
[in]guideCurvesAn optional parameter which specifies the curves used through which the shape of the lofted solid can be guided. Guide curves can be in any number but each of them should start from the first cross section, pass through each cross section and end at the last cross section.
[in]pathCurveAnother optional parameter through which the lofting operation can control the construction of the solid.
[out]isOpenif all of the cross-section curves are open this parameter is true
[out]isClosedif all of the cross-section curves are close this parameter is true
[out]isAllPlanarif all of the cross-section curves are planar this parameter is true
Returns
Acad::eOk if the parameter 1 and 3 is valid or invalidInput otherwise
