Loads the module file of 'Application' as defined in DemandLoad portion of the CAD system's Windows Registry section.

Parameters
[in]applicationlogical name of the Application, as defined in DemandLoad Registry part of CAD system
[in]reasonbit flags that must match the 'Applications' DemandLoad flags in Windows Registry
[in]printdetermines whether the status message is printed or not
[in]asCmddetermines whether the Application's module file is loaded "as Command" or a DLL file only
Returns
true, if Application's module file is successfully loaded, false otherwise
Remarks
at least one of the several AcadApp::LoadReasons flags defined in the ' DemandLoad Registry for 'Application' must matched by reason to load the module
