AcGeVector3d &  planeNormal,
bool &  isClosed,
double &  pathLength,

Verifies whether the given sweep entity is valid for sweeping operations.

Parameters
[in]pEntity- matrix of coordinate system
[out]entPlanarity- returns the planarity, if the sweep entity is valid
[out]planePnt- returns the sweep entity's plane origin (for a planar entity), or the entity start point (for a linear entity)
[out]planeNormal- returns the sweep entity's plane normal (for a planar entity), or the entity vector (for a linear entity)
[out]isClosed- true, if the entity uses a closed curve representation
[out]pathLength- the approx. length of the entity's curve
[in]showMessage- specifies whether to print an error message
Returns
Acad::eOk if the sweep entity is valid, error code otherwise
