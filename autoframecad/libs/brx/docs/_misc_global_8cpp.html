This function reads an UTF-8 character from given file.

Parameters
[in]pFilefile to read from
[out]widechwide character containing the UTF-8 character read
[in]chFmtrcharacter formatter to be used
Returns
true for success, false for error conditions
Remarks
UTF-8 encodes a single character in 1 to 4 bytes. The bit patterns in the first byte determine the number of bytes used to encode the particular character : 0xxxxxxx (0x7F) Character that takes up one byte. 110xxxxx (0xC0) Character that takes up two bytes. 1110xxxx (0xE0) Character that takes up tree bytes. 11110xxx (0xF0) Character that takes up four bytes.

Updates the parameters of the text style table record (having the name specified by giTextStyle 's style name) with those set in giTextStyle.

Returns the object id of the text style table record.

Parameters
[in]giTextStyleobject from which to read text style parameters
[out]textStyleTRIdid of the text style table record which parameters are set
Remarks
If a text style table record with the searched name doesn't exist a new one will be created, else the existing one will be used.
If the style name of giTextStyle is set to NULL a new name will be generated for the newly created text style table record.
Returns
Acad::eOk if successful

UX_EXPORT_FUNCTION Acad::ErrorStatus toAcDbTextStyle ( AcGiTextStyle &  giTextStyle,
const ACHAR *  pName,
AcDbObjectId &  id

This function checks if there is any existing text style with the name passed to function and updates the information obtained from the AcGiTextStyle.

If there is no style having the name quired then a new text style record is created with the information obtained from the AcGiTextStyle object.

Parameters
[in]giTextStyleobject from which to read text style information
[in]pNamename of the text style name which will be checked or created
[out]idid of the text style table record for which the information will be set
Remarks
If a text style table record with the searched name doesn't exist a new one will be created, else the existing one will be used.
If the style name of giTextStyle is set to NULL a new name will be generated for the newly created text style table record.
Returns
Acad::eOk if successful
