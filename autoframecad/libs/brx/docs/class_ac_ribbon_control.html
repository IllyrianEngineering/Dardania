◆ addTab()

AcRibbonUtils::AcResult AcRibbonControl::addTab ( AcRibbonTab *  pAcTab)
pure virtual

Appends a tab to Ribbon Control.

The tab content can be either empty or can contain panels.
Objects can be allocated on the heap or on the stack. Objects that are allocated on the stack are removed from the memory when they go out of scope. Objects allocated on the heap can be either deleted by the Caller or can be managed by the Application. Same goes for all the ribbon control objects (panel, items...etc).

Parameters
[in]pAcTabribbon tab to be added
Returns
AcRibbonUtils::eOk, or an error code otherwise
