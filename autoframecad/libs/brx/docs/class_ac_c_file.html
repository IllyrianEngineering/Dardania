◆ readBOM()

UX_EXPORT_FUNCTION bool AcCFile::readBOM ( )

This functions read the ByteOrderMask (BOM) from begin of file, and adjusts the current file format, if a valid BOM is found.

Must be used when file position is at the beginning of the file.

Returns
true, if a valid BOM was read and the file format has been adjusted; false, if there is no valid BOM at begin of file - in this case, file format setting is not changed, and file position pointer is reset to begin of file

◆ writeBOM()

UX_EXPORT_FUNCTION bool AcCFile::writeBOM ( )

This function writes the ByteOrderMask (BOM), corresponding to current file format setting, to begin of file; for kAnsi file format, nothing is written to file.

Returns
true for success, false otherwise
Remarks
the BOM must always be written to begin of file ! if this file uses kAnsi or kUnknown format, nothing is written, and 'false' is returned
