◆ setKeywordList()

UX_EXPORT_FUNCTION void AcEdJig::setKeywordList ( const ACHAR *  newList)

Sets a copy of passed newList as the keyword list for the following AcEdJig::acquireXX calls.

The keyword list can contain up to 30 space delimited keywords. Sample lists:
"ONE,O TWO,TW THREE,TH" - list with abbreviations repeated after comma.
"One Two THree" - abbreviations given as upper case.
"Eins Zwei Drei _ One Two THree" - local keywords followed by underscore and respective global keywords.

Parameters
[in]newListof keywords
