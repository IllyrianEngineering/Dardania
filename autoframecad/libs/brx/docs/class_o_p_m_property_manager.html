◆ catType()

unsigned int OPMPropertyManager::catType ( )

Returns whether the wrapper class supports IOPMPropertyExtensionImpl or IOPMPropertyExtensionImpl2 (and IAcPiCategorizePropertiesImpl).

The only differences in interfaces is, that the first choice implements ICategorizeProperties and the 2nd one IAcPiCategorizeProperties (which has all the functions ICategorizeProperties has and some more). Returns 2 for IAcPiCategorizeProperties supported, 1 otherwise

◆ createPropertyManager()

OPMPropertyManager * OPMPropertyManager::createPropertyManager ( AcRxClass *  pAcRxClass)
static

The only supported way to create a property manager.

Creates a COM Object OPMPropertyManager with reference count of 0. The CLSID is the sole connection from the property manager to the custom entity wrapper. With help of the AcRxClass the CLSID can be retrieved.
