◆ applyFeatures()

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmFlangeBase::applyFeatures ( )
virtual

This function finally creates the SheetMetal features on the result entities, after "run()" call; should be called only after successful cal to "run()" function, result entities can be used for previewing.

NOTE 1 : Result generation is split into two stages :

  1. Solid generation, can be used for previewing
  2. Feature generation, same solids are used as provided by getResult()

NOTE 2 : getResult() should be called before applyFeatures() !

Returns
Acad::eOk if successful, error status otherwise
