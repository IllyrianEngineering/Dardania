UX_EXPORT_FUNCTION Acad::ErrorStatus AcDbDictionaryWithDefault::getObjectBirthVersion ( AcDb::AcDbDwgVersion &  majorVersion,
AcDb::MaintenanceReleaseVersion &  minorVersion 
)
virtual

Returns the drawing and maintenance release versions into which this object must be stored.

Parameters
[out]majorVersionmajor version
[out]minorVersionminor version
Returns
Acad::eOk
Remarks
This function returns the same value as the corresponding functions of other CAD libraries for the parameter majorVersion, but it returns another value for parameter minorVersion. Other CAD libraries may return kMRelease37 and this function returns kMRelease42 same as the underlying DwgDirect function does. In questions of versioning we take the majorVersion information of the really underlying implementation.
