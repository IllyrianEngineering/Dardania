◆ applyFeatures()

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmFlangeBend::applyFeatures ( AcDb3dSolid *  pSolid)
virtual

This function finally creates the SheetMetal features on the target entity, after "run()" call; should be called only after successful cal to "run()" function, result entities can be used for previewing.

NOTE 1 : Result generation is split into two stages :

  1. Solid generation, can be used for previewing
  2. Feature generation, same solids are used as provided by applyResult(); the result entity has to be already added to database

NOTE 2 : applyResult() should be called before applyFeatures() !

Parameters
[in]pSolid- the target 3dSolid entity to apply the SheetMetal feature to
Returns
Acad::eOk if successful, error status otherwise
