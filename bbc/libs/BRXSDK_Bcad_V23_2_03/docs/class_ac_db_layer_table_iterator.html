UX_EXPORT_FUNCTION bool AcDbLayerTableIterator::getSkipHidden ( ) const

Returns iterators aktual way to handle hidden layers.

Returns
true, if hidden layer table records will be skipped
false if iterator will iterate on hidden records.
Remarks
the SkipHidden value only has effect on platforms, that support hidden layers. Bricscad does not support hidden layers.
