◆ hasOverriddenProperties()

UX_EXPORT_FUNCTION bool BrxComponentInstance::hasOverriddenProperties ( ) const

Indicates whether some of component properties (set for component definition) have been overridden for this instance.

I.e., this component instance has own values for some built-in component properties. If you need to extract them, you must obtain BrxComponentDefinition object using the block reference of this instance. E.g., the code below shows how to check sectionability for the given instance @instance attached to a block reference. BrxComponentDefinition component = BrxComponentDefinition::getComponentDefinition(instance.blockReferenceId()); bool isSectionable = component.isSectionable();

Returns
bool - true, if values for some of build-in properties have been overridden for this BrxComponentInstance object otherwise
