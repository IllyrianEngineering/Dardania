◆ OnGetTipSupport()

UX_EXPORT_FUNCTION ADUI_REPLY CAdUiEdit::OnGetTipSupport ( CPoint &  pnt)
protectedvirtual

This function is called to determine which kind of tip window is supported.

Parameters
[in]pntthe cursor's current position, given in screen coordinates
Returns
kAdUiReply_Nothing : if no kind of tip window is supported kAdUiReply_DrawTip : for ownder-drawn controls supporting owner-drawn text tips kAdUiReply_TextTip : the control supports text tips kAdUiReply_ToolTip : the control only supports default ToolTips
Note
any CAdUiEdit derived classes should support TextTips, so kAdUiReply_TextTip is the default return
