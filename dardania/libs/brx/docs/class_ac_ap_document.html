◆ popDbmod()

UX_EXPORT_FUNCTION Acad::ErrorStatus AcApDocument::popDbmod ( )

Restores the Drawing variable DBMOD with the topmost stack value and removes this value from the stack.

Must be preceeded by a pairing call of pushDbmod

Returns
Acad::eIllegalReplacement, if the stack was empty Acad::eInvalidContext, if the document is not yet ready for DBMOD Acad::eOk if successful
Remarks
Works only for Active document.

◆ pushDbmod()

UX_EXPORT_FUNCTION void AcApDocument::pushDbmod ( )

Stores the DBMOD value for this document on a stack.

So changes can be made by application and if then DBMOD is restored, the database is in unchanged state, if it was before. Has to be paired with popDbmod.

Remarks
Works only for Active document.
