◆ applyFeatures()

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmSplit::applyFeatures ( AcDb3dSolid *  pSolid)
virtual

This function finally creates the SheetMetal features on the target entity, after "run()" call; should be called only after successful cal to "run()" function, result entities can be used for previewing.

NOTE 1 : Result generation is split into two stages :

  1. Solid generation, can be used for previewing
  2. Feature generation, same solids are used as provided by applyResult(); the result entity has to be already added to database

NOTE 2 : applyResult() should be called before applyFeatures() !

Parameters
[in]pSolid- the target 3dSolid entity to apply the SheetMetal feature to
Returns
Acad::eOk if successful, error status otherwise

◆ applyResult()

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmSplit::applyResult ( AcDb3dSolid *  pSolid)
virtual

This function applies the result of given type to the given solid; should be called only after successful cal to "run()" function, before calling applyFeatures().

NOTE : see description for applyFeatures() function

NOTE : for performance reason, the result is moved to the provided solid; therefore, it is not possible to apply the same result twice !

Parameters
[in]pSolid- the target 3dSolid entity to apply the result to
Returns
Acad::eOk if successful, error status otherwise

◆ convertBendToJunction()

UX_EXPORT_FUNCTION bool BrxSmSplit::convertBendToJunction ( ) const
virtual

Returns the option whether one of the bends will be converted to junction in case of line intersection, after split; if true, one of the bends adjacent to selected flange created by splitting will be converted to junction, if it is adjacent to bend relief feature if false, bends will never be converted to junctions.

NOTE : this option is applicable only to flange face splitting

Returns
bool - true, if conversion into junction is used, false otherwise
