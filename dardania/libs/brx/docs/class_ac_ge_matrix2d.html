UX_EXPORT_FUNCTION Adesk::Boolean AcGeMatrix2d::isSingular ( const AcGeTol &  acTol = AcGeContext::gTol) const

Returns Adesk::kTrue if and only if this matrix is singular.

A matrix is singular if and only if its determinant = 0. A singular matrix cannot be inverted. In a singular matrix the coordinate vector are parallel or one of them has a length of 0.

Parameters
