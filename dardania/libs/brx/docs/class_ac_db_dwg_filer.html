Detailed Description

_

Binary filer for drawings.

Remarks
Keep care of writing and reading the objects using the same type. E.g writing an AcDbObjectId as AcDbSoftOwnershipId and reading it in as AcDbSoftPointerId will lead to unpredictable results up to crash.
