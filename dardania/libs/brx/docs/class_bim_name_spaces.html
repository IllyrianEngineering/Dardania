static

Creates a new NameSpace with given name and label, as visible or invisible, for the specified database.

Parameters
[in]szName- the name for the new NameSpace
[in]szLabel- the label for the new NameSpace; if empty or NULL, the name string is used as label
[in]visible- true, to set the NameSpace visible; false to set it invisible
[in]pDb- database to be used; if not specified, or NULL, the current drawing database is used
Returns
BimApi::eOk, if successful; error status otherwise (i.e. BimApi::eNameSpaceAlreadyExists)
