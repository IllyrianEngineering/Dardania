◆ setBuildLeftOutBodies()

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmRepair::setBuildLeftOutBodies ( double  bodyVolumeRatioThreshold)
virtual

Defines a threshold for the volume of small bodies to be erased by repair operation; sometimes after healing, some significant pieces of initial body are remaining; by default, operation doesn't search for it, but it may search all the left out bodies with volume > bodyVolumeRatioThreshold * initialVolume; if volume of left out bodies is less than bodyVolumeRatioThreshold * bodyVolume, bodies will be erased.

Parameters
[in]bodyVolumeRatioThreshold- ratio/percentage for the volume of small bodies to be erased
Returns
Acad::eOk if successful, error status otherwise

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmRepair::setSideFace ( const AcDbFullSubentPath &  face)
virtual

Defines the side face used by the repair operation; the repair operation expects to receive a single face related to flange or lofted bend feature; the side will be used as primary during thickness repairing; repair operation accepts exactly one of these inputs types :

  • flange or lofted bend face, wrong bend face
  • or hard edge for local repair

    Returns
    Acad::eOk if successful, error status otherwise
