UX_EXPORT_FUNCTION bool acdbDisplayPreviewFromDwg ( const ACHAR *  pDrawingFile,
void *  pDisplayWindow,
const Adesk::UInt32 *  pBackground 
)

Generates the preview bitmap of the drawing contained by pDrawingFile.

If pBackground is not NULL then the returned preview image's background will have the color specified by this parameter.

Parameters
[in]pDrawingFileThe name (full path) of the file containing the drawing to generate the preview from
[in]pDisplayWindowThe handle (HWND) of the window to display the preview image
[in]pBackgroundThe background color (in RGB format) of the preview image.
Returns
true if successful false otherwise
Remarks
If the pDisplayWindow window is larger or smaller than the image, then the image will be stretched to fit.
