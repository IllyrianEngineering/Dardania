UX_EXPORT_FUNCTION AcRibbonUtils::AcResult AcRibbonPanelSource::addItem ( AcRibbonItem *  pAcItem)

Appends an item to this panel source.

Panel source content can be organized in multiple rows using row panels and row breaks. There are also separators which can be added between items to separate them.
Objects can be allocated on the heap or on the stack. Objects that are allocated on the stack are removed from memory when they go out of scope. Objects allocated on the heap can be either deleted by the Caller or can be managed by the Application.

Parameters
[in]pAcItemitem to be added
Returns
AcRibbonUtils::eOk, or an error code otherwise
