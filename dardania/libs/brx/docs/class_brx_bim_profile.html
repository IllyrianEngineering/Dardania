const double  rotationAngle = 0.0,
const bool  useAlignment = true 
)
static

Applies the specified Profile to a set of points, creating an AcArray of (DBR or NDBR) AcDb3dSolid* entities.

Parameters
[in]points- ordered array of points; any two adjacent points are the start and end of line to which profile will be applied
[in]profile- the profile to be applied
[in]pDb- optional AcDbDatabase*, if non-NULL, the generated profile elements (and connectors) will be added to that database
[in]connectorType- type of connector that will be used
[in]rotationAngle- optional rotation (in radians) of the profile (default 0); Profile plane is plane that has its normal codirectional to the line to which this profile is applied, x-axis and y-axis will be calculated using transformation that transform z-axis of WCS to normal
[in]useAlignment- controls the alignment of created solids one by one (default : true)
Returns
BimApi::eOk for success, or error status
