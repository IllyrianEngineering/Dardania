UX_EXPORT_FUNCTION void AcDbDate::setJulianDay ( long  day)

Sets the day as Julian day number.

Works correctly as date from 2378862(1.1.1801) to 2488069(31.12.2099) In fact setting with this function and getting with the get function julianDay work for all possible numbers. But the correct date result is only given for the numbers in above range
