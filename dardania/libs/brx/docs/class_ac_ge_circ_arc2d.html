UX_EXPORT_FUNCTION AcGeCircArc2d::AcGeCircArc2d ( const AcGePoint2d &  ptStart,
const AcGePoint2d &  ptEnd,
double  bulge,
Adesk::Boolean  bulgeType = Adesk::kTrue 
)

Creates an arc with given start and end points, and specific bulge.

Direction of the arc is counterclockwise. Start and end points must be different. If bulgeType is Adesk::kTrue, then bulge is the maximum distance between the arc and the chord between start and end points. If bulgeType is Adesk::kFalse, then bulge is the tangent of 1/4 of the angle of the arc segment between the start and end points.

Parameters
[in]ptStartof the arc
[in]ptEndof the arc
[in]bulgeof the arc
[in]bulgeTypespecifies how bulge should be interpreted
